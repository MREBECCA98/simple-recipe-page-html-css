Simple Recipe Page - HTML & CSS

Questa è una semplice pagina di ricetta per un piatto chiamato "Focaccia with Caramelized Onions, Cranberries, and Brie", creata utilizzando solo HTML e CSS.
Descrizione:
Il progetto mostra una pagina di ricetta con una struttura ben definita che include:
-Un'intestazione con un logo e un breve sottotitolo.
-Una sezione principale con il nome della ricetta, una descrizione e un'immagine.
-Un elenco degli ingredienti richiesti per la ricetta.
-Una sezione con il metodo di preparazione.
-Un footer con un link a una ricetta correlata.

La pagina è progettata con un layout responsive per adattarsi bene su schermi di diverse dimensioni.

Tecnologie Utilizzate:
-HTML5 per la struttura della pagina.
-CSS3 per lo styling e il layout.
